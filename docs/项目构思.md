## 功能
1. 某个文件夹内有许多的子文件夹，每个子文件夹内是一本书籍的多种格式；子文件夹名字，例 “十四五”新发展格局-中国经济行稳致远（套装共10册） - 刘世锦" 设为 info
2. 通过ai分析 info, 获取 书籍名字 name, 书籍作者 author ,书籍类型 type
3. 按照type生成一个新的类型文件夹, 然后将此子文件夹拷贝到 新的类型文件夹中，同时修改子文件夹名字为 书籍名字name；最后设文件夹中的图片路径为 pic
4. 更新json文件,如果没有就新增；记录数据格式合理(你想下要如何组织)
5. 每一个子文件夹都按如上步骤执行
