## 现状分析

### 已有环境变量（已在 .env 中声明）

| 变量 | 文件 | 默认值 | 说明 |
|------|------|--------|------|
| `AI_BASE_URL` | analyzer.ts | 无（必填） | AI API 地址 |
| `AI_API_KEY` | analyzer.ts | 无（必填） | API 密钥 |
| `AI_MODEL` | analyzer.ts | 无（必填） | 模型名称 |
| `AI_CONCURRENCY` | analyzer.ts | 3 | 并发数 |
| `AI_VERIFY` | analyzer.ts | true | 核查模式 |
| `QINIU_*` (5个) | uploaders/qiniu.ts, qiniu/client.ts | 见文件 | 七牛云配置 |

### 硬编码常量（适合提取为环境变量）

| 常量 | 文件:行 | 当前值 | 建议 |
|------|---------|--------|------|
| `BATCH_SIZE` | analyzer.ts:5 | 60 | 提取为 `AI_BATCH_SIZE`，影响 token 消耗 |
| `temperature` | analyzer.ts:67 | 0.1 | 提取为 `AI_TEMPERATURE`，影响输出稳定性 |
| `MAX_RETRIES` | analyzer.ts:3 | 3 | 提取为 `AI_MAX_RETRIES` |
| `RETRY_DELAY_MS` | analyzer.ts:4 | 2000 | 提取为 `AI_RETRY_DELAY` |
| 图片最大宽度 | uploader.ts:12 | 1200 | 提取为 `UPLOAD_MAX_WIDTH` |
| JPEG 质量 | uploader.ts:13 | 80 | 提取为 `UPLOAD_JPEG_QUALITY` |

### 适合保留为代码常量的（不建议提取）

| 常量 | 原因 |
|------|------|
| 图片扩展名列表 `scanner.ts:5` | 业务规则，非配置 |
| Qiniu 批量操作限制 `bucket-manager.ts` | API 上限，非用户可调 |
| 自动保存间隔 `upload-pics.ts:57` | 内部优化，非用户关注 |
| MD5 哈希截取长度 `database.ts:27` | 文件命名约定，改了会破坏 |
| Excel 列宽 `export-excel.ts` | 纯展示，非配置 |
| 远程 key 前缀 `upload-pics.ts:50` | 存储路径约定，改了会破坏 |

## 开发方案

### 要实现什么

1. 新建 `src/config.ts` 统一管理所有配置，集中读取环境变量 + 提供默认值 + 注释说明
2. 将 6 个硬编码常量提取为环境变量
3. 各模块改为从 config.ts 读取配置，消除分散的 `process.env` 调用
4. 更新 `.env.example` 和 `.env` 添加新变量和注释

### 与现有功能的关系

- 纯重构，不改变任何业务逻辑和默认行为
- 所有默认值保持不变，不影响现有 .env 配置

### 需要修改的文件

```
src/analyzer.ts      # 改为从 config 读取 AI 配置，移除 process.env 调用
src/uploader.ts      # 改为从 config 读取图片压缩参数
src/uploaders/qiniu.ts  # 改为从 config 读取七牛云配置
src/qiniu/client.ts  # 改为从 config 读取七牛云配置
.env.example         # 添加新变量 + 注释
.env                 # 添加新变量
```

### 需要新增的文件

## 实施步骤

**步骤 1：创建 config.ts**

- [ ] 新建 `src/config.ts`，集中定义所有配置项

```typescript
import * as path from 'path';
import * as fs from 'fs';
import 'dotenv/config';

// AI 分析配置
export const ai = {
  get baseUrl() { return process.env.AI_BASE_URL; },
  get apiKey() { return process.env.AI_API_KEY; },
  get model() { return process.env.AI_MODEL; },
  get concurrency() { return Math.max(1, parseInt(process.env.AI_CONCURRENCY || '', 10) || 3); },
  get verify() { return process.env.AI_VERIFY !== 'false'; },
  get batchSize() { return parseInt(process.env.AI_BATCH_SIZE || '', 10) || 60; },
  get temperature() { return parseFloat(process.env.AI_TEMPERATURE || '') || 0.1; },
  get maxRetries() { return parseInt(process.env.AI_MAX_RETRIES || '', 10) || 3; },
  get retryDelay() { return parseInt(process.env.AI_RETRY_DELAY || '', 10) || 2000; },
};

// 图片上传配置
export const upload = {
  get maxWidth() { return parseInt(process.env.UPLOAD_MAX_WIDTH || '', 10) || 1200; },
  get jpegQuality() { return parseInt(process.env.UPLOAD_JPEG_QUALITY || '', 10) || 80; },
};

// 七牛云配置
export const qiniu = {
  get accessKey() { return process.env.QINIU_ACCESS_KEY; },
  get secretKey() { return process.env.QINIU_SECRET_KEY; },
  get bucket() { return process.env.QINIU_BUCKET; },
  get domain() { return process.env.QINIU_DOMAIN; },
  get zone() { return process.env.QINIU_ZONE || 'z0'; },
};
```

## 约束与风险

- **兼容性**：所有新增环境变量都有默认值，不设则行为不变，零破坏性
- **性能**：getter 方式读取，每次访问都会解析，但调用频率极低无影响
- **安全**：.env 文件已在 .gitignore 中，但当前 .env 被追踪，需注意不要泄露密钥
