## 实施步骤

**步骤 1：更新 README.md**

- [ ] 在「快速开始」section 的开发模式之后、「核心功能」之前，新增操作流程

新增内容要点：

**第 1 步：整理书籍**（自动）
- 命令：`npm run dev -- tidy --input D:\原始书籍 --output D:\图书合集_done`
- 注意：生成 `books.json` 数据库；自动去重跳过已处理书籍

**第 2 步：上传封面图片**（自动）
- 命令：`npm run dev -- upload-pics --db D:\图书合集_done\books.json --output D:\图书合集_done`
- 注意：每 5 本自动保存，支持断点续传

**第 3 步：上传百度云盘**（手动）
- 原因：百度云盘 API 限制，生成分享链接需付费
- 操作：上传文件夹 → 按类型批量生成链接 → 导出 CSV
- 注意：CSV 文件名必须为类型名（如 `传记.csv`），保存到 books.json 同级目录

**第 4 步：导入分享链接**（自动）
- 命令：`npm run dev -- import-links --dir D:\图书合集_done`
- 注意：自动保存 JSON 副本到 `result/` 目录

**第 5 步：导出 Excel**（自动）
- 命令：`npm run dev -- export-excel --db D:\图书合集_done\books.json`
- 注意：输出到 `result/` 目录
