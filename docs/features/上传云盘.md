## 前置条件

1. 在 [百度网盘开放平台](https://pan.baidu.com/union/) 注册开发者账号、创建应用、获取 AppKey + AppSecret
2. 申请**上传权限**（接入地址：https://pan.baidu.com/union/apply/ ）
3. 购买**分享服务**（创建分享链接为付费接口，详见平台"计费说明"）
4. 应用上传路径固定为 `/apps/{申请接入的产品名称}/`，用户在网盘中看到的是 `/我的应用数据/{产品名称}/`

## 开发方案

### 要实现什么

- 核心功能 1：`auth-baidu` 命令，完成 OAuth2 授权并存储 token 到本地
- 核心功能 2：`upload-baidu` 命令，将整理后的书籍文件夹批量上传到百度网盘
- 核心功能 3：上传完成后为每本书的文件夹创建永久分享链接
- 核心功能 4：`BookInfo` 新增 `linkUrl` 字段，记录分享链接；已上传书籍自动跳过

### 与现有功能的关系

- 参考模式：`src/upload-pics.ts`（遍历书籍、过滤已上传、周期性保存、重试）
- 依赖模块：`src/database.ts`（loadDatabase、saveDatabase）、`src/types.ts`（BookInfo）
- 集成位置：`src/index.ts` 新增 `auth-baidu` 和 `upload-baidu` 命令

### 百度网盘 API 概览

| 操作 | 接口 | 说明 |
|------|------|------|
| OAuth2 授权 | `GET /oauth/2.0/authorize` | authorization_code 模式，redirect_uri=oob |
| 获取 Token | `POST /oauth/2.0/token` | code 换 token，或 refresh_token 刷新 |
| 创建文件夹 | `POST /rest/2.0/xpan/file?method=create` | isdir=1 |
| 预上传 | `POST /rest/2.0/xpan/file?method=precreate` | 计算分片 MD5，获取 uploadid |
| 分片上传 | `POST /rest/2.0/pcs/superfile2?method=upload` | 4MB 分片，multipart/form-data |
| 创建文件 | `POST /rest/2.0/xpan/file?method=create` | 合并分片，返回 fs_id |
| 创建分享 | `POST /apaas/1.0/share/set?product=netdisk` | 付费接口，返回 link + pwd |

### 上传策略

```
对每本书：
1. 创建远程文件夹 /apps/{appName}/{rootDir}/{type}/{name}/（已存在则忽略）
2. 读取本地文件夹下所有文件
3. 逐文件上传：
   a. precreate（传入 content-md5 + slice-md5）→ 如果 return_type=2 说明文件已存在，跳过
   b. 分片上传缺失的 block（4MB 一片，图片通常只需一片）
   c. create 合并分片
4. 获取文件夹 fs_id（从创建文件夹响应或文件列表 API）
5. 创建永久分享链接（period=0）
6. 更新数据库 linkUrl
```

- **重复文件处理**：precreate 传入 `content-md5` + `slice-md5`，API 自动识别已有文件（return_type=2），无需重复上传
- **重复书籍跳过**：`linkUrl` 非空则跳过
- **批量防护**：每本书上传完毕后间隔 500ms，每 5 本自动保存数据库
- **Token 管理**：存储在 `~/.books-tidy/baidu-pan-token.json`，过期前 5 分钟自动刷新

```typescript
export function updateBookLinkUrl(
  db: BooksDatabase,
  bookId: string,
  linkUrl: string,
): void {
  const book = db.books.find((b) => b.id === bookId);
  if (book) {
    book.linkUrl = linkUrl;
  }
}
```

- [ ] 获取文件信息（`getFileInfo(path)` — 返回 fs_id）
- [ ] 创建分享链接（`createShare(fsidList, period, pwd)` — 返回 link）

关键实现细节：

```
OAuth2 授权流程：
1. 构造授权 URL: https://openapi.baidu.com/oauth/2.0/authorize?response_type=code&client_id={APP_KEY}&redirect_uri=oob&scope=basic,netdisk
2. 打印 URL，提示用户在浏览器打开并授权
3. 用户输入授权后页面显示的 code
4. POST https://openapi.baidu.com/oauth/2.0/token 换取 access_token + refresh_token
5. 保存到 ~/.books-tidy/baidu-pan-token.json

文件上传流程：
1. 读取文件，计算 content-md5（整个文件）和 slice-md5（前 256KB）
2. 将文件按 4MB 分片，计算每个分片的 MD5
3. POST precreate: path, size, isdir=0, block_list=[md5数组], content-md5, slice-md5, rtype=3
   → return_type=2: 文件已存在，跳过后续步骤
   → return_type=1: 获取 uploadid 和需要上传的 block_list
4. 对每个需要上传的分片: POST superfile2/upload (multipart/form-data)
5. POST create: path, size, isdir=0, uploadid, block_list=[md5数组], rtype=3

创建文件夹:
POST precreate: path, size=0, isdir=1, autoinit=1
POST create: path, size=0, isdir=1
（如果 errno=-8 表示已存在，忽略即可）

创建分享:
POST /apaas/1.0/share/set: fsid_list=[fs_id], period=0(永久), pwd=随机4位
返回: data.link（分享链接）, data.pwd（提取码）
linkUrl 格式: "https://pan.baidu.com/s/xxxx 提取码: abcd"
```

**步骤 4：上传命令编排器（src/upload-baidu.ts）**

- [ ] 实现 `uploadBaiduCommand(options)` 函数

```typescript
export async function uploadBaiduCommand(options: UploadBaiduOptions): Promise<void> {
  const dbPath = path.resolve(options.db);
  const outputDir = path.resolve(options.output);
  const rootDir = options.rootDir || '图书合集';

  // 1. 加载客户端（自动处理 token）
  const client = new BaiduPanClient();

  // 2. 加载数据库，过滤待上传书籍
  const db = loadDatabase(dbPath);
  const pending = db.books.filter((b) => !b.linkUrl);
  const skipped = db.books.length - pending.length;
  console.log(`共 ${db.books.length} 本书籍，${skipped} 本已有 linkUrl 跳过，${pending.length} 本待上传`);

  if (pending.length === 0) {
    console.log('没有需要上传的书籍');
    return;
  }

  let successCount = 0;
  let failCount = 0;
  const failedBooks: string[] = [];

  for (let i = 0; i < pending.length; i++) {
    const book = pending[i];
    console.log(`[${i + 1}/${pending.length}] ${book.name}`);

    try {
      // a. 获取本地书籍文件夹路径和文件列表
      const bookDir = path.join(outputDir, path.dirname(book.pic));
      const files = fs.readdirSync(bookDir).filter(f => fs.statSync(path.join(bookDir, f)).isFile());

      // b. 创建远程文件夹
      const remoteDir = `/apps/${client.appName}/${rootDir}/${book.type}/${book.name}`;
      await client.createFolder(remoteDir);

      // c. 逐文件上传
      for (const file of files) {
        const localPath = path.join(bookDir, file);
        const remotePath = `${remoteDir}/${file}`;
        await client.uploadFile(localPath, remotePath);
      }

      // d. 获取文件夹 fs_id 并创建分享
      const folderInfo = await client.getFileInfo(remoteDir);
      const shareResult = await client.createShare([folderInfo.fs_id], 0);

      // e. 更新数据库
      const linkUrl = `${shareResult.link} 提取码: ${shareResult.pwd}`;
      updateBookLinkUrl(db, book.id, linkUrl);
      successCount++;
      console.log(`  ✓ ${linkUrl}`);

      // f. 周期性保存
      if ((i + 1) % 5 === 0) {
        saveDatabase(dbPath, db);
        console.log('  (已自动保存)');
      }

      // g. 间隔，避免限频
      await sleep(500);
    } catch (err) {
      failCount++;
      failedBooks.push(book.name);
      console.error(`  ✗ 上传失败: ${(err as Error).message}`);
    }
  }

  saveDatabase(dbPath, db);
  console.log('\n--- 上传完成 ---');
  console.log(`成功: ${successCount} | 失败: ${failCount} | 跳过: ${skipped}`);
  if (failedBooks.length > 0) {
    console.log(`失败列表: ${failedBooks.join(', ')}`);
  }
}
```

## 约束与风险

- **上传路径限制**：百度网盘 API 限制文件只能上传到 `/apps/{appName}/` 目录下，用户在网盘中看到的是 `/我的应用数据/{appName}/`，无法指定到其他位置
- **分享接口付费**：创建分享链接为付费接口，需在百度网盘开放平台购买分享服务
- **Token 有效期**：access_token 约 30 天有效，refresh_token 约 10 年有效；客户端自动刷新，但需首次授权
- **API 限频**：百度网盘 API 有频率限制，批量上传 400-500 本预计耗时 2-4 小时（每本间隔 500ms）
- **无新依赖**：全部使用 Node.js 内置 `https` 模块，手动构造 multipart/form-data 请求体
- **图片文件**：书籍文件夹中的图片通常小于 4MB，多数情况下只需一个分片，上传效率较高
