## 3. 开发方案

### 3.1 要实现什么

1. 读取 books.json 数据库，按 `type` 字段分组
2. 生成分类型 JSON 文件 + 汇总 index.json，保存到 `result/configs/`
3. 将 `result/configs/` 下所有文件上传到七牛云（`books-tidy/` 前缀）

### 3.2 与现有功能的关系

- 复用 `loadDatabase()`（database.ts）读取 books.json
- 复用 `QiniuUploader`（uploaders/qiniu.ts）上传文件，七牛云同名 key 自动覆盖
- 模式参考 `export-excel.ts`：读 DB → 分组 → 输出到 result/

### 3.3 文件清单

- 新增：`src/generate-data.ts` — generate-data 命令编排器
- 修改：`src/index.ts` — 新增 generate-data 命令注册

### 3.4 新增依赖

无。

处理流程：
1. 加载数据库（`loadDatabase(dbPath)`）
2. 按 `book.type` 分组到 `Map<string, BookInfo[]>`
3. 生成 `index.json`：`[{ type, count }]` 数组，按 count 降序
4. 遍历 Map，每个类型写入 `{type}.json`
5. 实例化 `QiniuUploader`，遍历 `outputDir` 下所有文件，逐个上传
6. 输出生成和上传统计
