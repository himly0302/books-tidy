## 目标
批量上传本地图片到图床，返回 URL 并更新 JSON

## 本地数据

某本书处理后的数据如下
```
{
    "id": "fc93b2f9-3513-48f3-bb9e-058bfffd2fc0",
    "name": "世界经典学术集系列",
    "author": "列夫•马诺维奇",
    "type": "学术",
    "pic": "学术/世界经典学术集系列/9f368d20.jpg",
    "addedAt": "2026-04-29T14:14:26.643Z"
}
```

注: 整体的json文件和type文件夹同级

## 效用的效果

1. 自行筛选合适的图床, 让我做决定是否使用或提供相关配置信息

2. 上传本地图片返回 图床网络连接 url
- json对象新增字段 picUrl: url

3. 在保证图片质量的前提下，上传到图床的图片尽量压缩大小

4. 果json对象中有 picUrl 字段(不为空)，则跳过此图片的上传逻辑
